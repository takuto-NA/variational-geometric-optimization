本章は FEM（有限要素法）/ PDE の応用例を収録する。

## VGO の文脈での見方（FEM = 離散化された変分最適化）

PDE を「方程式を解く問題」としてだけでなく、**汎関数 \(J[u]\) の停留条件（変分原理）**として捉えると、FEM は VGO の典型例になる。

- **目的関数**: 物理・幾何に由来するエネルギー（例: Poisson なら Dirichlet energy）や、弱形式に対応する汎関数 \(J[u]\)
- **変数（最適化変数）**: 連続場 \(u(x)\)（関数空間上の変数）  
  FEM では基底展開 \(u_h(x)=\sum_i q_i \phi_i(x)\) により、係数 \(q\)（節点自由度）へ落ちる
- **制約**: 境界条件（Dirichlet/Neumann）、領域・材料・接触など（VGO の制約付き最適化として扱える）
- **幾何（計量・前処理）**: 関数空間の内積（\(L^2\), \(H^1\)）が、離散化後は質量行列・剛性行列として現れ、更新則（勾配法/自然勾配/前処理）の「形」を決める

この見方により、FEM の線形方程式 \(Kq=f\) は「離散汎関数の最適性条件（\(\nabla J_h(q)=0\)）」として統一的に理解でき、以降の章で扱う **制約**・**幾何に沿った更新**・**実装パターン** と自然に接続できる。

Remark:
上の「FEM = 変分最適化」は、少なくとも **対称（自己随伴）な問題**（Poisson、線形弾性、拡散など）では、Ritz の最小化と弱形式が一致する、という意味で強い。
一方で、対流項などにより **非対称** になる場合は、一般の Galerkin（重み付き残差）として定式化され、必ずしも「あるスカラー汎関数の最小化」そのものにはならない。
VGO 的には、この差は「純粋な \(\mathcal F\) の最小化」に閉じるか、あるいは外力/非保存項（\(J\) や散逸 \(G\) を含む一般形）として扱うか、という Functional/Geometry 側の設計として整理できる。
